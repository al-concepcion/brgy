# Official Signatures

Place signature images in this folder with the following names:

## For Certificates:
- `captain.png` - Signature of Hon. Ivan Adal (Punong Barangay)
- `kagawad1.png` - Signature of Hon. Yasser Alapag
- `kagawad2.png` - Signature of Hon. Dave Abellanosa
- `kagawad3.png` - Signature of Hon. Hadden Abogado
- `kagawad4.png` - Signature of Hon. Minsey Alfaro

## Image Requirements:
- Format: PNG with transparent background (recommended)
- Size: Maximum 200x40 pixels for captain, 150x40 for kagawads
- Color: Black or dark blue ink
- Quality: High resolution, clear and legible

## How to Create Signature Images:
1. Sign on white paper with black pen
2. Scan at 300 DPI or take high-quality photo
3. Use image editing software to:
   - Crop to signature only
   - Remove background (make transparent)
   - Save as PNG
4. Upload to this folder

## Fallback:
If signature images are not available, the documents will show empty signature lines for manual signing.
